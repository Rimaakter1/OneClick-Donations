What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree structure. It allows programming languages like JavaScript to interact with, access, and manipulate elements, attributes, and content on a web page dynamically.

How do you select an element from the DOM?

I can select an element from the DOM using some methods. such as :

  1. document.getElementById() method selects an element by its id
  2. document.getElementsByClassName() method select elements by class name
  3. document.getElementsByTagName() select elements by tag name
  4. document.querySelector() selects the first element that matches a CSS selector
  5. document.querySelectorAll() selects all elements that match a given CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a way to manage events in JavaScript by adding one event listener to a parent element instead of adding listeners to each child element. When a child element is clicked, the event bubbles up to the parent, which can then handle the event.
It is useful because the method reduces the number of event listeners needed, works well with dynamic content, and keeps the code cleaner and simpler.

How do you manipulate an element's attributes and styles using the DOM?

I will manipulate an element's attributes and styles using the DOM with the following methods:

For manipulate and element's attributes:

  1. getAttribute(): To retrieve the value of an attribute.
  2. setAttribute():To change the value of an attribute or add a new one.
  3. removeAttribute() : to remove an attribute.

For manipulate and element's styles:

  1. Modify inline styles with element.style (e.g., element.style.color = 'blue').
  2. Add or remove CSS classes with classList.add(), classList.remove()
